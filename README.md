# Tasks-APP #

#### React TasksApp ####

* TasksApp
* 0.1.0

##### Requirements:  
- npm needs to be installed.


### Setting Up 
----

##### 1. Clone the repositorie  
> git clone ggit@github.com:augustoscher/tasks-app.git

##### 2. Access the repositorie
> cd tasks-app

##### 3. Install the dependencies
> sudo npm install

##### 4. Running
> npm start


#### 5. Test
Open http://localhost:3000
